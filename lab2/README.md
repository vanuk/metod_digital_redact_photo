# Лабораторна робота 2 (Python)

## Тема
Читання, відображення та збереження відео за допомогою OpenCV.

## Підготовка
Встановіть OpenCV (у вашому віртуальному середовищі):

```bash
pip install opencv-python
```

## Завдання 1
Читання та відображення відеофайлу (процедурний підхід).

```bash
python lab2/task1.py --video lab2/input.mp4
```

## Завдання 2
Читання, відображення та збереження потоку з вебкамери (процедурний підхід).

```bash
python lab2/task2.py --camera-index 0 --output lab2/webcam_output.avi --fps 20
```

## Завдання 3
Читання та відображення відеофайлу (ООП).

```bash
python lab2/task3.py --video lab2/input.mp4
```

## Завдання 4
Читання, відображення та збереження потоку з вебкамери (ООП).

```bash
python lab2/task4.py --camera-index 0 --output lab2/webcam_output_oop.avi --fps 20
```

## Керування
У всіх програмах натисніть `q`, щоб завершити роботу.
